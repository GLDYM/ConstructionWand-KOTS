# Construction Wand KOTS

The Construction Wand - KOTS Fork, add more mod compat and QoL Change.

This version introduces the following changes:

- Replaces the original **Sneak + "wand config key"** operation with just the **"wand config key"** **(default: Left Ctrl**) for actions such as undo, mode switching, core switching, and opening the GUI.
- Supports using items from the armor slots and Curios accessory slots.
- Supports using items from L2 Backpacks.
- Supports using items from Applied Energistics 2: Portable cells, Wireless terminals, and Wireless crafting terminals.
- Supports Tom’s Simple Storage wireless terminal, with access logic identical to the original.
- Supports Refined Storage (RS) wireless terminal, consuming energy based on item count and RS configuration, while allowing unlimited distance and cross-dimensional access.
- Enables nested reading for Shulker Boxes, Sophisticated Backpacks, and L2 Backpacks.
